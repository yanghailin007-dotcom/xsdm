### 四、JSON格式验证测试

#### 4.1 标准格式
```json
{
  "title": "测试",
  "episode": 1,
  "description": "测试描述",
  "protagonist": {
    "name": "测试",
    "appearance": "测试"
  }
}
```
- [ ] 验证通过
- [ ] 可正常提交

#### 4.2 缺少必填字段
```json
{
  "title": "测试"
}
```
- [ ] 验证失败
- [ ] 提示"缺少description字段"

#### 4.3 完整分镜格式
```json
{
  "title": "测试",
  "description": "测试",
  "protagonist": {"name": "测试", "appearance": "测试"},
  "shots": [{"shot_number": 1, "content": "测试"}]
}
```
- [ ] 验证显示"导入分镜"模式
- [ ] 直接使用提供的分镜
