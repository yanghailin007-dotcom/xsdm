# 🚀 大文娱系统 - 启动说明

## 快速开始（两种入口）

### 1️⃣ 初始化安装（首次使用）

**适用场景：** 第一次使用、环境损坏、给其他新用户

```bash
# 方式一：双击运行（推荐Windows用户）
初始化安装.bat

# 方式二：命令行
python setup.py
```

**功能：**
- 自动检查 Python 环境
- 如无 Python，自动下载嵌入式 Python 3.11（约 11MB）
- 安装 pip 和项目依赖
- 自动修复代码兼容性问题
- 安装完成后自动启动服务

---

### 2️⃣ 启动服务（日常使用）

**适用场景：** 日常开发、快速启动

```bash
# 方式一：双击运行（推荐Windows用户）
启动服务.bat

# 方式二：命令行
python start.py
```

**功能：**
- 快速检查环境
- 自动清理端口 5000
- 启动 Web 服务
- 自动打开浏览器访问首页

---

## 文件说明

| 文件 | 用途 |
|------|------|
| `setup.py` | 初始化安装脚本 |
| `start.py` | 日常启动脚本 |
| `初始化安装.bat` | Windows 批处理入口（初始化） |
| `启动服务.bat` | Windows 批处理入口（日常） |
| `python-embed/` | 嵌入式 Python 环境（自动创建） |

---

## 访问地址

服务启动后访问：

- 🏠 **首页**: http://localhost:5000/landing
- ✍️ **小说创作**: http://localhost:5000/
- 🔌 **API**: http://localhost:5000/api

---

## 常见问题

### Q: 提示找不到 Python？
运行 `初始化安装.bat`，会自动下载安装嵌入式 Python。

### Q: 端口 5000 被占用？
脚本会自动清理，无需手动操作。

### Q: 依赖安装失败？
检查网络连接，重新运行 `初始化安装.bat`。

### Q: 如何安装新依赖？
```bash
python-embed\python.exe -m pip install <包名>
```
