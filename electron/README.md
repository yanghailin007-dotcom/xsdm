# 短剧工作室 - Windows桌面版

## 📦 开发者打包指南

### 环境要求
- Node.js 16+ (https://nodejs.org/)
- Python 3.8+ (https://www.python.org/)
- Git (可选)

### 打包步骤

1. **安装Electron依赖**
   ```bash
   cd electron
   npm install
   ```

2. **一键打包**
   ```bash
   # 双击运行
   build_all.bat

   # 或在命令行运行
   cd electron
   build_all.bat
   ```

3. **获取安装包**
   - 打包完成后，安装包位于: `electron/dist/短剧工作室-Setup-1.0.0.exe`
   - 大小约: 200-300MB

### 开发模式

如果你想在开发模式下测试:

```bash
cd electron
dev.bat
```

这将启动Flask后端和Electron前端，方便调试。

## 👥 用户安装指南

### 系统要求
- Windows 10/11 (64位)
- 4GB+ 内存
- 2GB+ 可用磁盘空间

### 安装步骤

1. **下载安装包**
   - 下载 `短剧工作室-Setup-1.0.0.exe`

2. **运行安装程序**
   - 双击安装包
   - 选择安装路径
   - 点击"安装"

3. **首次启动配置**
   - 启动应用后，点击"⚙️ VeO配置"
   - 输入你的VeO API密钥
   - 点击"保存配置"

4. **开始使用**
   - 创建新项目
   - 导入小说
   - 生成视频

### 获取API密钥

**VeO API:**
1. 访问 https://jyapi.ai-wx.cn
2. 注册/登录账号
3. 获取API Key

### 常见问题

**Q: 启动后显示"无法连接到服务器"**
A: 请检查防火墙设置，允许应用访问网络

**Q: 视频生成失败**
A:
- 检查API密钥是否正确
- 检查网络连接
- 查看错误信息

**Q: 如何更新版本**
A: 下载新版本安装包，直接安装即可覆盖旧版本

**Q: 如何卸载**
A: 控制面板 → 程序和功能 → 卸载"短剧工作室"

### 数据存储位置

- **项目文件**: `C:\Users\你的用户名\AppData\Local\短剧工作室\projects`
- **配置文件**: 浏览器LocalStorage
- **日志文件**: `C:\Users\你的用户名\AppData\Local\短剧工作室\logs`
