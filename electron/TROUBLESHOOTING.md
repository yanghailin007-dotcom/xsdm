# ⚠️ 打包前必读 - 解决权限问题

## 问题说明

Electron Builder在Windows上打包时需要创建符号链接，这需要特殊权限。

错误信息：`ERROR: Cannot create symbolic link : 客户端没有所需的特权`

## 解决方案（选择其一）

### 方案1：启用Windows开发者模式（推荐）

1. 打开 **Windows设置**
2. 进入 **更新和安全** → **开发者选项**
3. 启用 **开发人员模式**
4. 重启电脑
5. 重新运行打包命令

### 方案2：以管理员身份运行

1. 右键点击 **命令提示符** 或 **PowerShell**
2. 选择 **以管理员身份运行**
3. 切换到项目目录：
   ```cmd
   cd D:\work6.06\electron
   ```
4. 运行打包命令：
   ```cmd
   npm run build:dir
   ```

### 方案3：使用简化打包（临时方案）

如果上述方案都不可行，我们可以先测试开发模式：

```cmd
cd electron
npm run dev
```

这会直接启动应用，不需要打包。

## 打包后的测试步骤

打包成功后：

1. 进入 `electron/dist/win-unpacked/`
2. 双击 `短剧工作室.exe`
3. 应用会自动启动Flask后端
4. 测试各项功能

## 注意事项

- 第一次打包会下载Electron二进制文件（约100MB），需要等待
- 打包过程可能需要5-10分钟
- 确保网络连接正常
