# 🚀 一键部署工具 - 最终使用说明

## ✨ 推荐使用

**[`一键部署_自动版.bat`](一键部署_自动版.bat)** - 真正的一键部署

### 自动完成所有操作

只需双击运行，脚本会自动完成：

1. ✅ **同步所有代码**
   - Python后端代码
   - 前端UI文件（CSS、JS、HTML）
   - 配置文件
   - 所有项目文件

2. ✅ **初始化日志系统**
   - 创建日志目录
   - 初始化日志文件
   - 配置日志记录

3. ✅ **重启服务**
   - 自动停止旧服务
   - 启动新服务
   - 验证服务状态

4. ✅ **验证部署**
   - 检查服务是否运行
   - 显示访问地址
   - 提供日志查看方式

---

## 📝 使用方法

### 日常部署（推荐）

```bash
# 双击运行
scripts/deploy/一键部署_自动版.bat
```

**然后强制刷新浏览器**：
- Windows: `Ctrl + F5`
- Mac: `Cmd + Shift + R`

### 如果UI仍未更新

清除浏览器缓存：
1. 按 `F12` 打开开发者工具
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

或使用无痕模式访问

## 💡 重要提示

1. **首次部署会较慢**（需要安装依赖）
2. **后续部署很快**（只同步修改的文件）
3. **部署后务必强制刷新浏览器**
4. **如果UI没更新，清除浏览器缓存**

## ❓ 常见问题

### Q: 部署后UI没有更新？
A: 强制刷新浏览器（Ctrl+F5），或清除浏览器缓存

### Q: 服务无法访问？
A: 运行 `scripts/deploy/view_server_logs.bat` 选择"4. 错误日志"查看问题

### Q: 如何查看详细日志？
A: 运行 `scripts/deploy/view_server_logs.bat` 选择相应的日志选项
