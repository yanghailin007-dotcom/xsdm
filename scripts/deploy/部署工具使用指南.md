# 小说生成系统 - 部署工具使用指南

## 📋 概述

本项目提供了一键部署脚本，用于将本地代码（包括UI部分）同步到服务器并重启Web服务。

## 🚀 快速开始

### 方法一：完整部署（推荐）

使用最新版本的完整部署脚本：

```bash
scripts\deploy\一键部署_完整版.bat
```

**功能：**
- ✓ 同步所有代码（src、web、config、scripts）
- ✓ 同步UI文件（web/static/js、web/static/css、web/templates）
- ✓ 重启Web服务
- ✓ 验证部署状态

**包含的UI文件：**
- 所有JavaScript文件 (*.js)
- 所有CSS样式文件 (*.css)
- 所有HTML模板 (*.html)

### 方法二：快速重启

只需重启服务，不同步代码：

```bash
scripts\deploy\一键部署_终极版.bat
```

### 方法三：检查部署状态

查看服务器当前状态：

```bash
scripts\deploy\check_deploy_status.bat
```

## 📁 部署内容

### 同步的目录

```
src/              # 核心业务代码
web/              # Web应用和UI文件
  ├── static/     # 静态资源
  │   ├── js/     # JavaScript文件（UI逻辑）
  │   └── css/    # CSS样式文件
  ├── templates/  # HTML模板
  └── api/        # API接口

```

## 📊 部署流程

1. **环境检查**
   - 检查私钥文件
   - 检查项目目录
   - 测试SSH连接

2. **代码同步**
   - 同步 src/
   - 同步 web/（包含所有UI文件）
   - 同步 config/
   - 同步 requirements.txt

3. **服务器部署**
   - 停止旧服务
   - 备份旧代码
   - 部署新代码
   - 更新依赖

4. **启动服务**
   - 启动Gunicorn服务
   - 验证服务状态

5. **验证部署**
   - 测试Web响应
   - 显示访问信息

## ⚠️ 重要提示

### 浏览器缓存

部署后如果UI没有更新，请：

1. **强制刷新浏览器**
   - Windows: `Ctrl + F5` 或 `Ctrl + Shift + R`
   - Mac: `Cmd + Shift + R`

2. **清除浏览器缓存**
   - 打开开发者工具 (F12)
   - 右键点击刷新按钮
   - 选择"清空缓存并硬性重新加载"

3. **清除特定资源缓存**
   ```javascript
   // 在浏览器控制台执行
   location.reload(true);
   ```

### UI更新确认

部署后确认UI文件已更新：

```bash
ssh -i "d:\work6.05\xsdm.pem" root@8.163.37.124 "ls -lh /home/novelapp/novel-system/web/static/js/"
```

检查文件时间戳是否为最新。

## 🐛 故障排查

### 问题1: SSH连接失败

**原因：** 密钥权限不正确

**解决：**
```bash
icacls "d:\work6.05\xsdm.pem" /inheritance:r
icacls "d:\work6.05\xsdm.pem" /grant:r "%USERNAME%:F"
```

### 问题2: 服务启动失败

**检查：**
```bash
ssh -i "d:\work6.05\xsdm.pem" root@8.163.37.124 "tail -50 /home/novelapp/novel-system/logs/error.log"
```

### 问题3: 端口被占用

**清理端口：**
```bash
ssh -i "d:\work6.05\xsdm.pem" root@8.163.37.124 "lsof -ti:8080 | xargs kill -9"
```

### 问题4: UI没有更新

**确认文件同步：**
```bash
ssh -i "d:\work6.05\xsdm.pem" root@8.163.37.124 "ls -lt /home/novelapp/novel-system/web/static/js/ | head -10"
```

**确认浏览器缓存已清除**
