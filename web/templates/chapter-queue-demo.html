📖 使用方法
// 1. 初始化队列（比如生成20章，从第1章开始） chapterQueue.initQueue(20, 1); // 2. 更新章节状态（生成时自动有动画效果） // 状态选项: 'pending'(等待) | 'generating'(生成中) | 'completed'(完成) | 'error'(失败) chapterQueue.updateChapterStatus(1, 'generating', 50); // 第1章生成中，50% chapterQueue.updateChapterStatus(1, 'completed'); // 第1章完成 chapterQueue.updateChapterStatus(2, 'generating', 30); // 第2章开始生成 // 3. 获取总进度 const progress = chapterQueue.getProgressPercentage(); console.log(`完成进度: ${progress}%`);
✨ 设计特点
  • 🎨 流水线式布局 - 章节像流水线上的产品一样从左到右移动
  • 🎯 颜色状态系统 - 灰色(等待) → 蓝色(生成中) → 绿色(完成)
  • ⚡ 动态流光效果 - 生成中的章节有脉冲动画，连接线有流动效果
  • 📊 实时统计 - 顶部显示等待/生成中/完成的章节数量
  • 🎯 智能滚动 - 自动滚动到当前生成的章节
  • 📱 响应式设计 - 适配移动端和桌面端
