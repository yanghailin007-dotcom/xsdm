# 视频生成系统 - 快速开始指南

## 🎬 三分钟上手

### 第一步：访问页面
```
http://localhost:5000/video-generation
```

### 第二步：选择小说
从左侧列表选择已完成第一阶段设定的小说

### 第三步：选择视频类型
点击三种视频类型之一：
- 🎬 **短片/动画电影** (5-30分钟) - 精选核心事件，艺术化镜头
- 📺 **长篇剧集** (20-40分钟/集) - 保留完整内容，适合连载
- 📱 **短视频系列** (1-3分钟) - 高光时刻，快速节奏

### 第四步：开始转换
点击"开始转换"按钮，等待完成

### 第五步：查看结果
转换完成后可以：
- 查看详细分镜头脚本
- 导出Markdown格式
- 转换其他小说或重新选择类型

---

## 🔄 导航说明

### 顶部导航栏
- **📚 小说项目** - 返回小说列表
- **🎬 视频生成** - 当前页面（重新开始）
- **🏠 控制台** - 返回主控制台

### 转换页面按钮

**配置阶段**：
- **← 选择其他小说** - 回到欢迎页面，保持当前选中的小说
- **← 重新选择类型** - 回到欢迎页面，清除所有选择

**结果阶段**：
- **📚 转换其他小说** - 选择不同的小说进行转换
- **🎬 重新选择类型** - 为当前小说选择不同的视频类型

---

## 📝 API快速调用

### 转换为长篇剧集
```bash
curl -X POST http://localhost:5000/api/video/convert \
  -H "Content-Type: application/json" \
  -d '{
    "title": "我的小说",
    "video_type": "long_series",
    "total_units": 10
  }'
```

### 预览第1集
```bash
curl http://localhost:5000/api/video/preview/我的小说/long_series/1
```

### 导出Markdown
```bash
curl http://localhost:5000/api/video/export/我的小说/long_series/1
```

## 💡 使用建议

### 选择建议

**短片/动画电影** - 适合：
- 动画短片制作
- 电影预告片
- 独立动画项目
- 需要强艺术感的作品

**长篇剧集** - 适合：
- 网络动画连载
- 电视动画系列
- 番剧制作
- 需要完整叙事的作品

**短视频系列** - 适合：
- 抖音/快手内容
- B站短视频
- 社交媒体营销
- 快速传播内容

### 最佳实践

1. **确保小说完整**：先完成第一阶段设定生成
2. **章节数量充足**：长剧集建议100章以上
3. **事件系统完整**：确保包含重大事件数据
4. **测试不同类型**：同一种小说可以尝试不同视频类型

## ❓ 常见问题

**Q: 如何返回到小说项目列表？**
A: 点击顶部导航栏的"📚 小说项目"

**Q: 转换完成后想换一种视频类型怎么办？**
A: 点击"🎬 重新选择类型"按钮，保持当前小说，重新选择视频类型

**Q: 想为不同的小说生成视频怎么办？**
A: 点击"📚 转换其他小说"按钮

**Q: 如何查看之前生成的分镜头脚本？**
A: 重新选择相同的小说和类型，转换会覆盖之前的结果（未来版本将支持历史记录）
