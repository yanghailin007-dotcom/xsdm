## 迁移步骤

对于每个页面，按以下步骤操作：

1. **备份**
   ```bash
   cp web/templates/page.html web/templates/page.html.backup
   ```

2. **参考示例**
   复制 `web/templates/_template-example.html` 作为起点

3. **提取内容**
   - 找到页面主体内容（通常在 `<body>` 内，除去导航栏）
   - 提取页面特有的 CSS 和 JS

4. **应用模板**
   ```html
   {% extends "layouts/base.html" %}
   
   {% block title %}页面标题{% endblock %}
   
   {% block content %}
   <!-- 页面内容 -->
   {% endblock %}
   ```

5. **测试验证**
   - [ ] 页面正常渲染
   - [ ] 导航栏显示正确
   - [ ] 用户菜单可以展开/收起
   - [ ] 所有功能正常工作

6. **更新状态表**
   更新本文件中的迁移状态
