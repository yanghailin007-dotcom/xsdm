# 创意系统改进

## 快速开始

### 1. 同人小说检测已修复

**问题**：原创创意因包含功法名（如《吞天魔功》）被误判为同人小说

**解决**：现在只有明确包含同人标记才会触发检测

```python
# ✅ 会被识别为同人
"基于《凡人修仙传》的同人小说"
"改编自《火影忍者》"

# ❌ 不会被识别为同人（原创）
"主角修炼《吞天魔功》"
"获得《九转金身》功法"
```

### 2. 多文件创意存储

**问题**：所有创意在一个文件中，不便管理

**解决**：支持多文件独立存储

```
data/creative_ideas/
├── index.json              # 索引
├── 001_魔剑.json          # 创意1
├── 002_重生.json          # 创意2
└── ...
```

### 3. 迁移到多文件

```bash
python tools/migrate_creative_to_multi_files.py
```

自动备份、拆分、创建索引，无需手动操作。
