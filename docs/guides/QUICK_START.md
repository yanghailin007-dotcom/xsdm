## 🔧 常见修改

### 改变端口
编辑 web_server.py 最后:
```python
app.run(host='0.0.0.0', port=8000)  # 改成 8000
```

### 改变颜色
编辑 static/css/style.css:
```css
--primary-color: #667eea;  /* 改成其他颜色 */
```

### 改变标题
编辑 templates/index.html 或 novel_view.html 的 `<title>` 标签

## 🌐 API 速查 (给开发者)

```bash
# 健康检查
curl http://localhost:5000/api/health

# 生成小说
curl -X POST http://localhost:5000/api/generate-chapters \
  -H "Content-Type: application/json" \
  -d '{"chapters_count": 5}'

# 获取章节
curl http://localhost:5000/api/chapter/1

# 导出数据
curl http://localhost:5000/api/export-json > data.json
```
