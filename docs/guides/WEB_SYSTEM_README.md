# 🎨 小说生成系统 - Web 可视化平台

## 概述

这是一个完整的网页展示系统，允许您像浏览小说网站一样查看 AI 生成的小说内容，同时实时查看设计信息、质量评估等关键数据。

## 功能特性

### 📖 阅读页面 (`/novel`)

**三列布局设计：**

```
┌─────────────────────────────────────────────────────┐
│           小说生成系统                              │
└─────────────────────────────────────────────────────┘
┌──────────┬──────────────────────┬──────────┐
│  设计    │    正文内容          │  评估    │
│ 信息     │                      │ 数据     │
│          │  第 N 章标题         │          │
│ •标题    │  ================   │ ⭐ 评分  │
│ •进度    │                      │ ✓ 优点   │
│ •设定    │  生成的正文内容...  │ 💡 建议  │
│ •卖点    │  (字数、格式保留)    │ 📊 统计  │
│          │                      │          │
│ 📄 章节  │                      │          │
│ 列表     │                      │          │
│ ▼        │                      │          │
└──────────┴──────────────────────┴──────────┘
```

**功能：**
- ✅ 三列响应式布局
- ✅ 章节导航快速切换
- ✅ 实时质量评分显示
- ✅ 优缺点和改进建议
- ✅ 键盘快捷键支持（←→上下键切换章节）
- ✅ JSON 导出功能
- ✅ 打印功能

### 📊 仪表板 (`/dashboard`)

**实时统计面板：**
- 生成章节数统计
- 总字数统计
- 平均评分显示
- 生成进度百分比
- 章节详情表格
- 评分分布图表
- 最高/最低/平均指标

### 🏠 首页 (`/`)

**功能：**
- 产品特性展示
- 小说生成配置表单
- 实时生成进度显示
- 完成后自动跳转到阅读页面

## 快速开始

### 1. 安装依赖

```bash
pip install flask flask-cors
```

### 2. 启动服务

**方式一：直接运行**
```bash
cd d:\work6.03
python web_server.py
```

**方式二：使用启动脚本**
```bash
python start_web_server.py
```

### 3. 打开浏览器

访问 `http://localhost:5000`

## 使用流程

### 第一步：配置并生成

1. 在首页填写小说配置
   - 标题
   - 简介
   - 核心设定
   - 卖点
   - 总章节数
   - 本次生成数

2. 点击 `🚀 开始生成`

3. 等待生成完成（通常 1-2 秒）

### 第二步：查看阅读

1. 自动跳转到阅读页面 (`/novel`)

2. **左侧：查看设计信息**
   - 小说基本信息
   - 核心设定和卖点
   - 章节列表快速导航

3. **中间：阅读正文**
   - 点击章节快速切换
   - 使用键盘方向键导航
   - 保留原始格式

4. **右侧：查看质量评估**
   - 实时评分显示
   - 优点列表（绿色 ✓）
   - 改进建议（红色 ×）
   - 字数和生成时间

### 第三步：数据分析

1. 访问仪表板 (`/dashboard`)

2. 查看综合统计数据：
   - 章节统计
   - 评分分布
   - 字数分析
   - 进度追踪

3. 导出 JSON 数据供外部处理

## API 端点

### 首页和页面

| 端点 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 首页 |
| `/novel` | GET | 阅读页面 |
| `/dashboard` | GET | 仪表板 |

### 数据 API

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/health` | GET | 健康检查 |
| `/api/start-generation` | POST | 启动生成 |
| `/api/generate-chapters` | POST | 生成章节 |
| `/api/novel/summary` | GET | 获取小说摘要 |
| `/api/chapters` | GET | 获取章节列表 |
| `/api/chapter/<num>` | GET | 获取章节详情 |
| `/api/progress` | GET | 获取生成进度 |
| `/api/export-json` | GET | 导出完整 JSON |

## 常见问题

### Q: 如何修改监听端口？
**A:** 编辑 `web_server.py` 最后一行：
```python
app.run(host='0.0.0.0', port=8000)  # 改为 8000
```

### Q: 如何在其他机器上访问？
**A:** 使用服务器 IP：
```
http://<服务器IP>:5000
```

### Q: 如何自定义样式？
**A:** 编辑 `static/css/style.css`

### Q: 如何扩展功能？
**A:** 编辑相应的文件：
- 后端逻辑：`web_server.py`
- 前端页面：`templates/*.html`
- 前端脚本：`static/js/*.js`
- 样式表：`static/css/*.css`

## 遇到的常见问题及解决

### 问题 1：502 Bad Gateway

**原因：** Flask 服务未启动

**解决：** 
```bash
python web_server.py
```

### 问题 2：CORS 错误

**原因：** 跨域请求被阻止

**解决：** 已配置 flask-cors，应该不会出现。如有问题，检查 `web_server.py` 中的 CORS 配置。

### 问题 3：样式加载失败

**原因：** 静态文件路径错误

**解决：** 确保 `static/` 目录与 `web_server.py` 同级。
