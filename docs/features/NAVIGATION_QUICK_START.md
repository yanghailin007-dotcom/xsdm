## 🎮 使用新功能

### 在阅读页面看到:
```
┌─────────────────────────────┐
│    第 1 章 内容...         │
│    (正文)                  │
├─────────────────────────────┤
│  [下一章 →]    第 1 章   │  ← 导航条
└─────────────────────────────┘
```

### 三种方式切换章节

**方式 1: 点击按钮** ✅ 新功能
```
点击 "← 上一章" 或 "下一章 →"
立即跳转到上/下一章
```

**方式 2: 键盘快捷键** ✅ 原有功能
```
← / ↑  → 上一章
→ / ↓  → 下一章
```

**方式 3: 左侧章节列表** ✅ 原有功能
```
点击左侧的任意章节
直接跳转
```

## ❓ 常见问题

### Q: 按钮在哪里？
**A**: 在正文末尾，"第 N 章"指示器下方

### Q: 按钮什么时候隐藏？
**A**: 没有上一章时隐藏"上一章"，没有下一章时隐藏"下一章"

### Q: 点击按钮会发生什么？
**A**: 加载新章节，内容、评分、列表全部更新

### Q: 还能用键盘吗？
**A**: 可以，键盘快捷键仍然有效

### Q: 按钮可以自定义吗？
**A**: 可以，编辑 `templates/novel_view.html` 和 `static/css/style.css`

## 📝 修改说明

如果想进一步定制:

### 改变按钮文字
编辑 `templates/novel_view.html`:
```html
<!-- 改成想要的文字 -->
<button>上一篇 ◀</button>
<button>下一篇 ▶</button>
```

### 改变按钮样式
编辑 `static/css/style.css`:
```css
.chapter-navigation .btn {
    background-color: #your-color;
    /* 其他样式 */
}
```

### 改变按钮位置
在 `templates/novel_view.html` 中移动整个 `.chapter-navigation` 块
