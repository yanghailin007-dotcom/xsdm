```python
def _generate_expectation_mapping(self, writing_plan: Dict, stage_name: str) -> Dict:
    """为写作计划生成期待感映射（使用AI智能分析）"""
    
    # 1. 初始化期待感管理器
    expectation_manager = ExpectationManager()
    expectation_integrator = ExpectationIntegrator(expectation_manager)
    
    # 2. 获取重大事件
    major_events = event_system.get("major_events", [])
    
    # 3. 使用AI分析事件并添加期待感标签
    analysis_result = expectation_integrator.analyze_and_tag_events(
        major_events=major_events,
        stage_name=stage_name,
        api_client=self.generator.api_client,
        novel_title=self.generator.novel_data.get("novel_title", "")
    )
    
    # 4. 生成期待感映射并添加到写作计划
    expectation_map = expectation_manager.export_expectation_map()
    plan_container["expectation_map"] = expectation_map
    
    # 5. 保存期待感映射到项目目录
    expectation_map_file = project_dir / "expectation_map.json"
    with open(expectation_map_file, 'w', encoding='utf-8') as f:
        json.dump({
            'novel_title': novel_title,
            'stage_name': stage_name,
            'expectation_map': expectation_map,
            'generated_at': datetime.now().isoformat(),
            'total_tagged': total_tagged
        }, f, ensure_ascii=False, indent=2)
```

- ✅ 内嵌在 `stage_writing_plan` 的 `expectation_map` 字段

## Bug原因

1. **ProductLoader 未提取 expectation_map**
   - 只提取了 `major_events`、`stage_info`
   - 忽略了 `expectation_map` 字段

2. **Storyline产物中缺少 expectation_map**
   - `_extract_storyline_from_writing()` 方法没有保留 `expectation_map`
   - 提取后的 `storyline` 数据不包含期待感映射

3. **API使用了错误的生成方法**
   - 使用 `select_expectation_type()` 规则匹配
   - 而不是 `ExpectationIntegrator.analyze_and_tag_events()` AI分析

---

## 修复方案

### 方案1：修复ProductLoader，提取expectation_map

**修改文件**：`web/api/phase_generation_api.py`

在 `_extract_storyline_from_writing()` 方法中添加：


### 方案2：加载单独的expectation_map.json文件

在 `ProductLoader` 中添加新方法：

```python
def _load_expectation_maps(self, products):
    """加载所有阶段的期待感映射"""
    expectation_maps = {}
    
    # 从项目目录加载
    expectation_map_file = self.project_dir / "expectation_map.json"
    if expectation_map_file.exists():
        with open(expectation_map_file, 'r', encoding='utf-8') as f:
            data = json.load(f)
            stage_name = data.get('stage_name')
            expectation_map = data.get('expectation_map')
            if stage_name and expectation_map:
                expectation_maps[stage_name] = expectation_map
    
    return expectation_maps
```
