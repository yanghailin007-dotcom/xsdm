# 角色编辑器使用指南

## 概述

角色编辑器是一个用户友好的界面，用于管理和编辑小说项目中的角色信息。它取代了之前纯JSON展示的方式，提供了可视化的编辑体验。

## 功能特性

### 1. 角色列表管理
- 📋 左侧面板显示所有角色列表
- 🎨 每个角色卡片显示图标、名称、类型和描述
- ⭐ 支持快速浏览和选择角色

### 2. 角色信息编辑
右侧编辑面板分为以下几个部分：

#### 基本信息
- **角色名称**：角色的必填名称
- **角色类型**：可选择主角、配角、反派、导师、同伴等
- **角色图标**：12种预设图标可选
- **代表颜色**：8种主题颜色可选

#### 角色描述
- **外貌特征**：描述角色的外貌、穿着等
- **性格特点**：描述角色的性格、行为方式
- **背景故事**：描述角色的出身、经历等

#### 能力设定
- **特殊能力**：例如：御剑飞行、炼丹术
- **修炼等级**：例如：筑基期、金丹期
- **主要技能**：例如：剑术、阵法

#### 角色关系
- 动态添加/删除角色关系
- 支持多种关系类型：盟友、敌对、中立、家人、导师、朋友
- 可关联其他角色

### 3. 数据持久化
- ✅ 自动保存到服务器
- ✅ 实时更新角色网络视图
- ✅ 支持导出JSON格式

## 使用方法

### 打开角色编辑器

在项目可视化页面中：

1. 点击左侧菜单的"角色网络"按钮
2. 在角色关系网络视图右上角，点击"✏️ 编辑角色"按钮

### 创建新角色

1. 点击左侧面板顶部的"+ 新建角色"按钮
2. 系统会自动创建一个名为"新角色"的默认角色
3. 在右侧面板编辑角色信息
4. 点击"💾 保存角色"按钮保存

### 编辑现有角色

1. 在左侧列表中点击要编辑的角色卡片
2. 右侧面板会显示该角色的详细信息
3. 修改需要更改的字段
4. 点击"💾 保存角色"按钮保存更改

### 删除角色

1. 在角色卡片右上角点击🗑️图标
2. 确认删除操作

### 添加角色关系

1. 选中要编辑的角色
2. 在"角色关系"部分点击"+ 添加关系"
3. 选择关系类型（盟友、敌对等）
4. 输入相关角色的名称
5. 点击"💾 保存角色"

## 技术实现

### 文件结构

```
web/
├── templates/
│   └── components/
│       └── character-editor-modal.html  # 角色编辑器UI模板
├── static/
│   └── js/
│       └── character-editor.js          # 角色编辑器逻辑
└── api/
    └── character_api.py                 # 角色数据API
```

### API端点

#### 获取角色数据
```
GET /api/characters/{project_title}
```

#### 保存角色数据
```
POST /api/characters/{project_title}
Content-Type: application/json

{
  "characters": [...]
}
```

#### 导出角色数据
```
GET /api/characters/{project_title}/export
```

## 集成说明

### 在其他页面中使用角色编辑器

1. 引入JavaScript文件：
```html
<script src="/static/js/character-editor.js"></script>
```

2. 添加模态框容器：
```html
<div id="character-editor-modal"></div>
```

3. 调用打开函数：
```javascript
openCharacterEditor();
```

## 注意事项

1. **权限要求**：所有API端点都需要用户登录
2. **数据同步**：编辑器会自动同步到角色网络视图
3. **表单验证**：角色名称为必填项
4. **删除确认**：删除角色需要二次确认

## 故障排除

### 问题：编辑器无法打开
**解决方案**：检查浏览器控制台是否有错误，确保所有JavaScript文件已正确加载

### 问题：保存失败
**解决方案**：检查网络连接，确保服务器API正常运行，查看浏览器控制台的错误信息

### 问题：角色数据不显示
**解决方案**：确保项目已有角色数据，或通过API创建新角色
