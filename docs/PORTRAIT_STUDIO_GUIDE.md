# 人物剧照工作室 - 使用指南

## 🎨 概述

人物剧照工作室是一个专门的、独立的角色剧照生成工具，提供简洁的创作工作台界面。

## 🌟 主要特点

1. **独立界面**：不再是视频生成系统的子页面，而是专门的创作工具
2. **简洁工作流**：选择角色 → 编辑提示词 → 生成剧照
3. **智能提示词生成**：自动从小说角色数据生成详细提示词
4. **灵活自定义**：支持完全自定义角色描述
5. **参考图支持**：可上传参考图像以获得更一致的结果

## 📋 使用流程

### 方式一：小说角色模式

1. **访问工作室**
   - 登录系统后，访问 `/portrait-studio`
   - 或从首页点击"人物剧照工作室"

2. **选择小说角色模式**
   - 点击"📚 小说角色"卡片

3. **选择小说和角色**
   - 从下拉菜单选择您的小说项目
   - 点击角色卡片选择要生成剧照的角色
   - **自动进入工作台**（无需额外点击）

4. **编辑提示词**
   - 系统自动生成角色提示词
   - 可手动编辑提示词调整生成效果
   - 支持重新生成提示词
   - 可复制提示词到剪贴板

5. **上传参考图（可选）**
   - 拖拽图片或点击上传区域
   - 支持 PNG、JPG、WEBP 格式
   - 可随时移除或更换参考图

6. **设置生成参数**
   - **图片比例**：9:16（竖屏）、16:9（横屏）、1:1（方形）、4:3（标准）
   - **图片质量**：4K（超高清）、2K（高清）、1K（标清）
   - **艺术风格**：自动选择、写实风格、动漫风格、水墨风、赛博朋克、古风等

7. **生成剧照**
   - 点击"🎨 生成剧照"按钮
   - 等待生成完成（显示进度条）
   - 查看生成结果

8. **使用结果**
   - **下载**：保存生成的剧照
   - **用作参考图**：将当前剧照设为参考图，用于下次生成
   - **重新生成**：使用相同设置重新生成

### 方式二：自定义角色模式

1. **选择自定义角色模式**
   - 从主页点击"✨ 自定义角色"卡片

2. **填写角色信息**
   - **角色名称**（必填）：如"剑仙"、"魔尊"
   - **角色定位**（必填）：如"主角"、"反派"、"配角"
   - **外貌特征**（可选）：描述身高、体型、发型、眼睛等
   - **服装打扮**（可选）：描述服装、饰品、武器等
   - **性格特点**（可选）：如"冷静"、"热血"
   - **动作姿态**（可选）：如"站立"、"战斗"、"冥想"
   - **场景背景**（可选）：如"宫殿"、"森林"、"山巅"

3. **生成提示词并进入工作台**
   - 点击"📝 生成提示词并进入工作台"
   - 系统自动生成提示词并进入工作台页面

4. **后续步骤**
   - 与小说角色模式相同（编辑提示词、上传参考图、设置参数、生成）

## 💡 提示词编写技巧

### 基础要素

1. **角色描述**
   - 发色和发型
   - 眼睛颜色和形状
   - 皮肤和体型
   - 服装和配饰

2. **动作和表情**
   - 面部表情（微笑、凝视、愤怒）
   - 身体姿态（站立、坐、战斗）
   - 手部动作

3. **场景和背景**
   - 场景位置（宫殿、森林、战场）
   - 环境元素（云海、火焰、花朵）
   - 时间和天气（日落、雨天、夜晚）

4. **艺术风格**
   - 画风类型（写实、动漫、水墨）
   - 光影效果（柔光、强对比、侧光）
   - 色彩风格（温暖、冷色、鲜艳）

### 示例提示词

**仙侠风格角色**：
```
角色：剑仙
定位：主角

外貌特征：白发如雪，长发及腰，身穿白色仙袍，手持长剑，剑身散发出青色光芒
性格特点：清冷孤傲，眼神深邃，气质超凡

动作姿态：立于云端之上，手持长剑指向远方，衣袍随风飘扬
场景背景：云海翻涌，远处可见连绵的山峰，天空呈现淡紫色
艺术风格：水墨风，飘逸空灵，淡蓝和白色为主色调
```

**现代都市角色**：
```
角色：都市白领
定位：配角

外貌特征：黑色短发，戴着眼镜，穿着职业西装，手中拿着咖啡杯
性格特点：沉稳干练，眼神专注

动作姿态：站在办公室窗前，望向窗外城市景观
场景背景：现代化办公室，窗外是繁华的城市夜景，霓虹灯闪烁
艺术风格：写实风格，都市氛围感
```

## 🎯 最佳实践

### 提高生成质量

1. **使用参考图**：上传参考图可以获得更一致的角色外观
2. **详细描述**：提示词越详细，生成结果越符合预期
3. **指定风格**：明确艺术风格可以避免不确定的结果
4. **迭代优化**：使用"用作参考图"功能进行迭代改进

### 常见问题

**Q: 如何生成多个角色的合照？**
A: 在提示词中描述所有角色，例如："两个角色站立在一起，左侧是剑仙，右侧是魔尊..."

**Q: 如何调整角色的表情？**
A: 在提示词中添加表情描述，例如："嘴角带着微笑"、"眼神温柔"、"眉头微皱"

**Q: 生成结果不满意怎么办？**
A: 
1. 调整提示词描述
2. 上传参考图
3. 尝试不同的艺术风格
4. 使用"重新生成"按钮

**Q: 可以生成特定场景的角色吗？**
A: 可以，在提示词中详细描述场景，例如："角色站在盛开的樱花树下，花瓣飘落"
