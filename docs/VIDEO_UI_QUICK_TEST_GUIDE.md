## 问题排查

### 问题1: 进度条不动
**可能原因**:
- 后端返回 `in_progress` 状态但前端未识别
- 轮询间隔过长

**解决方法**:
```javascript
// 在浏览器控制台检查
console.log('API响应:', data);
// 确认status字段值
```

---

### 问题2: 视频无法加载
**可能原因**:
- 视频URL无效
- 网络连接问题
- 浏览器不支持格式

**解决方法**:
1. 检查浏览器控制台错误
2. 检查网络面板请求
3. 验证视频URL是否可访问

---

### 问题3: 控制按钮无响应
**可能原因**:
- JavaScript错误
- DOM元素未正确绑定
- CSS样式冲突

**解决方法**:
1. 刷新页面重试
2. 检查浏览器控制台错误
3. 验证事件监听器是否绑定
