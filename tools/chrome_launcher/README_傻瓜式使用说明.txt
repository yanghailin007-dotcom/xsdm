    大文娱 Chrome 启动器 - 傻瓜式使用说明
═══════════════════════════════════════════════════

📌 只需两步，轻松上手：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 第一步：双击运行 "一键启动.bat"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. 找到文件夹里的 "一键启动.bat" 文件
   2. 双击运行
   3. 等待自动完成（首次使用需要下载Chrome，约150MB）
   
   💡 如果没有Python，脚本会自动帮你安装！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 第二步：在浏览器中登录番茄账号
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. Chrome 会自动打开并跳转到番茄小说
   2. 用你的账号登录番茄
   3. 进入「作家专区」
   
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 第三步：回到大文娱网页点击「开始上传」
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. 保持 Chrome 窗口开着（不要关闭！）
   2. 回到大文娱网页
   3. 点击「检测连接」按钮
   4. 连接成功后，「开始上传」按钮会变为可用
   5. 选择要上传的小说，点击「开始上传」

═══════════════════════════════════════════════════
                     常见问题
═══════════════════════════════════════════════════

❓ 双击.bat后闪退怎么办？
   → 右键点击.bat文件，选择「以管理员身份运行」

❓ 提示"Python未安装"怎么办？
   → 选择选项 1，让脚本自动下载安装
   → 或者手动访问 python.org 下载安装

❓ Chrome下载很慢怎么办？
   → 检查网络连接
   → 如果是首次下载，耐心等待（约150MB）
   → 可以手动下载 Chrome 并放到 chrome/ 目录

❓ 提示"连接失败"怎么办？
   → 确保 Chrome 窗口没有关闭
   → 点击「检测连接」重试
   → 如果还是失败，关闭所有Chrome窗口，重新运行.bat

❓ 可以关闭启动器的黑窗口吗？
   → 可以！Chrome启动成功后，那个窗口可以关闭
   → 但千万不要关闭 Chrome 浏览器窗口
