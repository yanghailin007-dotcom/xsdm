# 落云宗"留守"双殊：韩长老不在家的日子 第1集
## 从创意到分镜完整制作方案

---

## 一、创意阶段

### 1.1 核心创意概念

**主题定位**：
- 类型：修仙武侠 + 女性成长 + 黑色幽默
- 核心矛盾：传统修仙文中"被留下的女人"视角
- 情感基调：笑中带泪，悲中见勇

**一句话故事**：
> 当修仙界最强"甩手掌柜"韩立闭关三十年，他的名义道侣慕沛灵和弟子程宣在落云宗艰难维持，表面抱怨吐槽，实则以命守护洞府，在一次次危机中证明"留守者"的价值。

**创意钩子（Hook）**：
- 反套路：不拍韩立的传奇，拍"留守者"的日常
- 共鸣点：现代职场"背锅侠"、"留守妻子"的修仙版映射
- 悬念点：三十年等待，等的人是否还会回来？

### 1.2 目标受众分析

| 维度 | 描述 |
|------|------|
| **核心受众** | 《凡人修仙传》原著粉、修仙题材爱好者 |
| **扩展受众** | 短剧观众、女性向内容消费者 |
| **年龄层** | 18-35岁 |
| **观看场景** | 移动端、碎片化时间、竖屏友好 |
| **内容偏好** | 强情绪、快节奏、有反转、能共情 |

### 1.3 差异化竞争点

与传统修仙短剧对比：

| 传统修仙短剧 | 本作品 |
|--------------|--------|
| 主角开挂升级 | 配角苦撑日常 |
| 男性视角主导 | 女性视角叙事 |
| 爽点密集 | 情感深度 + 黑色幽默 |
| 打斗为主 | 文戏武戏平衡 |

## 二、剧本阶段

### 2.1 角色设定

#### 慕沛灵（主角）

| 项目 | 内容 |
|------|------|
| **身份** | 韩立名义道侣，实际留守管理者 |
| **外貌** | 青色汉服，优雅端庄，眼角有疲惫细纹 |
| **性格** | 表面毒舌抱怨，内心坚韧忠诚，责任感极强 |
| **能力** | 结丹后期，被管理事务拖累修为停滞 |
| **痛点** | "师娘"空名 vs "看门人"现实，修为被低估 |
| **目标** | 守住洞府，等待韩立归来，证明自己 |
| **弧光** | 从被动承受 → 主动觉醒 → 以命守护 |

#### 程宣（配角）

| 项目 | 内容 |
|------|------|
| **身份** | 韩立弟子，跑腿传话 |
| **外貌** | 灰色弟子服，年轻慌张，永远手忙脚乱 |
| **性格** | 忠诚但能力不足，乐观但容易崩溃 |
| **作用** | 喜剧担当，衬托慕沛灵，提供信息 |
| **口头禅** | "师娘！又出事了！" |

#### 韩立（缺席的主角）

| 项目 | 内容 |
|------|------|
| **存在方式** | 不在场但处处在场（传音符、噬金虫、洞府规矩） |
| **角色功能** | 情感锚点（等待的对象）、压力来源（留下的烂摊子） |
| **悬念设置** | 三十年未归，是死是活？还会回来吗？ |

### 2.2 故事大纲（三幕结构）

#### 第一幕：建立（0-26秒）

**节拍设计**：
1. **开场画面**（0-5s）：落云宗全景，宏伟但空荡
2. **主题呈现**（5-14s）：慕沛灵抱怨日常，展示"留守"困境
3. **铺垫强化**（14-26s）：程宣汇报+噬金虫喂食，喜剧堆叠

**核心任务**：
- 建立世界观（修仙、洞府、韩立闭关）
- 建立人物关系（慕沛灵的管理者身份）
- 建立喜剧基调（表面抱怨的轻松感）

#### 第二幕：对抗（26-50秒）

**节拍设计**：
4. **转折点**（26-34s）：崖边对话，暴露真实情感
5. **游戏时间**（34-42s）：内心独白，揭示深层痛苦
6. **中点打击**（42-50s）：外敌入侵，危机降临

**核心任务**：
- 情感转折（从喜剧切入悲剧）
- 人物深度（展示"留守"的真正代价）
- 危机建立（外部威胁打破平衡）

#### 第三幕：高潮（50-76秒）

**节拍设计**：
7. **觉醒时刻**（50-56s）：慕沛灵决心守护
8. **激烈战斗**（56-64s）：以命相搏，展现实力
9. **结尾画面**（64-76s）：遥望星空，悬念钩子

**核心任务**：
- 人物弧光完成（从抱怨者到守护者）
- 主题升华（"留守"不是被动，是选择）
- 悬念设置（下一集预告）

### 3.2 关键镜头详细设计

#### 镜头2A：慕沛灵抱怨（中景）

**视觉要点**：
- 服装：青色丝绸汉服，略显凌乱（体现忙碌）

- 语气：抱怨但带点宠溺，像吐槽自家老公

```

```

## 四、制作执行指南

### 4.1 Veo视频生成流程

**输入格式**：
```json
{
  "prompt": "英文视觉描述（80-120词）",
  "duration": "5-8秒",
  "aspect_ratio": "16:9 或 9:16"
}
```

**生成策略**：
1. **分批生成**：按场景分批次，避免上下文丢失
2. **备选方案**：每个镜头生成2-3个版本，择优
3. **风格统一**：使用相同关键词（如"cyan silk Hanfu"）确保人物一致性

**提示词优化技巧**：
- ✅ 用 "flowing cyan silk Hanfu" 而不是 "beautiful costume"
- ✅ 用 "golden morning sunlight" 而不是 "good lighting"
- ✅ 用 "camera slowly pushing in" 而不是 "cinematic shot"
- ❌ 避免：epic, masterpiece, 4K, unreal engine, cinematic

### 4.3 剪辑建议

**节奏控制**：
```
前30秒：轻松喜剧 → 快剪，平均3-4秒/镜头
中间20秒：情感沉淀 → 慢剪，平均4-5秒/镜头  
后26秒：紧张战斗 → 快剪，平均3秒/镜头
结尾10秒：悬念留白 → 慢剪，5-7秒/镜头
```

**转场建议**：
- 场景2→3：硬切（喜剧节奏）
- 场景4→5：叠化（情绪转变）
- 场景6→7：闪白（危机降临）
- 场景9→10：慢 fade out（结尾回味）
